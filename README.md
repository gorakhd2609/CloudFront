# 🚀 AWS CloudFront CDN Project

This project demonstrates how to use Amazon CloudFront with S3 to deliver content globally with low latency.

---

## 📌 What is CloudFront?

CloudFront is a CDN that caches content at edge locations to provide faster delivery.

---

## ⚙️ Architecture

User → CloudFront → S3 Bucket

---

## 🔥 Features

- Global content delivery
- Caching mechanism (Cache Hit / Miss)
- Reduced latency
- Improved performance
